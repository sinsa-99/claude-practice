# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 프로젝트 개요

이 폴더는 Claude Code 학습, 블록체인(이더리움·솔리디티·STO·RWA) 개발 실습, 공인중개사 시험 대비 AI 도구 제작을 위한 작업 공간입니다.

---

## 행동 규칙 (반드시 준수)

### 언어
- **항상 한국어로 대답**한다. 코드 외 모든 설명·질문·답변은 한국어 사용.

### 개념 설명 방식
- 개념을 설명할 때 **반드시 비유를 포함**한다.
  - 예: "가스(Gas)는 자동차 기름값과 같아요. 더 복잡한 경로(연산)일수록 기름이 더 많이 듭니다."

### 코드 작성 규칙
- 작성하는 **모든 코드에 한국어 주석 필수**. 각 함수·로직의 목적을 주석으로 설명.
- 스마트 컨트랙트는 보안 취약점 검토 포함 (reentrancy, overflow 등).
- 프라이빗키·시드구문은 절대 하드코딩 금지 — `.env` 파일로 관리.

### 정보 신뢰도
- 출처가 불확실한 정보는 **사실처럼 단정하지 않는다**.
- 확실하지 않으면 "~로 알려져 있습니다", "공식 문서 확인이 필요합니다" 등으로 표현.
