# Claude Code 단축키 모음

## 터미널 단축키 (Claude Code 밖)

| 단축키 | 기능 |
|--------|------|
| `Cmd+T` | 새 터미널 탭 열기 |
| `Cmd+N` | 새 터미널 창 열기 |
| `Ctrl+C` | 실행 중인 것 강제 종료 |
| `Ctrl+V` | 붙여넣기 (터미널에서만 Ctrl!) |
| `Cmd+C` | 텍스트 복사 (드래그 후) |
| `↑ 화살표` | 이전에 입력한 명령어 불러오기 |
| `Ctrl+L` | 터미널 화면 깨끗하게 지우기 |

---

## Claude Code 안 단축키

| 단축키 | 기능 |
|--------|------|
| `Option+Enter` | 줄바꿈 (긴 내용 여러 줄 입력) |
| `Ctrl+C` | Claude 작업 즉시 중단 |
| `Ctrl+D` | Claude Code 종료 |
| `Esc Esc` | 직전 작업 되돌리기 |
| `Shift+Tab` | Plan Mode 전환 (실행 전 계획만 보기) |
| `Ctrl+B` | 백그라운드로 보내기 |
| `Ctrl+O` | 숨겨진 내용 펼쳐보기 |
| `?` | 단축키 전체 목록 보기 |

## 슬래시 명령어 (대화창에서 입력)

| 명령어 | 기능 |
|--------|------|
| `/init` | CLAUDE.md 자동 생성 (첫 시작 때 한 번) |
| `/compact` | 대화 압축 (자주 쓰기!) |
| `/clear` | 대화 완전 초기화 |
| `/cost` | 토큰 사용량 확인 |
| `/model` | 모델 변경 (Sonnet/Haiku/Opus) |
| `/effort high/low` | 생각 깊이 조절 |
| `/rename 이름` | 세션에 이름 붙이기 |
| `/memory` | CLAUDE.md 편집 |
| `/permissions` | 권한 설정 |
| `/plan` | Plan Mode 진입 |
| `/rewind` | 여러 단계 되돌리기 |
| `/context` | 컨텍스트 사용량 확인 |
| `/clone` | 현재 대화 복제 (실험용) |
| `/quit` | 종료 |
| `/help` | 전체 명령어 목록 |

## 터미널에서 Claude 시작할 때

| 명령어 | 기능 |
|--------|------|
| `claude` | 기본 시작 |
| `claude --continue` | 직전 세션 이어서 |
| `claude --resume` | 세션 목록에서 골라서 이어서 |
| `claude --resume 이름` | 이름으로 바로 불러오기 |
| `claude -p "질문"` | 대화 없이 답변만 출력 |
| `claude --model sonnet` | 모델 지정해서 시작 |

---

## 자주 쓸 TOP 5

1. `/compact` — 대화 길어지면 무조건
2. `Ctrl+C` — 뭔가 이상하면 즉시 중단
3. `Esc Esc` — 실수했을 때 되돌리기
4. `Option+Enter` — 긴 내용 여러 줄 입력
5. `/rename` — 중요한 세션 이름 붙여두기
